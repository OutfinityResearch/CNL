Syntax

Readable, fixed structures for deterministic parsing.

CNL-PL enforces explicit forms for noun phrases, boolean grouping, and triplet assertions. Every valid input yields a single AST.

Syntax highlights

  • Noun phrases start with determiners or quantifiers.
  • Mixed AND/OR chains require explicit grouping.
  • Relative clauses repeat the pronoun for each condition.

Example: structured relative clauses

A user who is active and who knows Python and whose score is greater than 10.

Example: explicit grouping

If either the user is an admin or the user is a dev, and the system is prod.

Example: pragmatics command

Return the name of every user whose role is Admin and who has logged in today.
