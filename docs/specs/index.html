Specs

Design specifications and living references.

The DS series captures vision, implementation plans, syntax, semantics, testing, and evaluation suites. Each runtime file mirrors a specification in docs/specs/src.

Spec structure

  • DS: Core design specs (vision to evaluation).
  • src: File-level mirrored design notes.
  • tests: Test plan and test file descriptions.
  • evals: Parsing and reasoning evaluation suites.
