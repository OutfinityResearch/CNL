CNL Pragmatics
Pragmatic Mode

Plan

Generate ordered action sequences to achieve goals.

Concept

The Plan pragmatic transforms CNL from a query language into an AI agent. Given a Goal State (defined as a condition), it searches for a sequence of Actions (defined in Action Blocks) that will transition the world from the current state to the goal state.

Syntax

Plan to achieve the system is active.

Execution Logic

The engine treats the Goal ("system is active") as the destination in a state-space graph.

  • It looks at available actions (e.g., StartSystem).
  • It checks preconditions: StartSystem requires Power is On.
  • It recurses: "Plan to achieve Power is On".
  • It finds TurnOnPower action.

The result is a linear plan: [TurnOnPower(), StartSystem()]. This uses ActionPlan IR and heuristic search (A* or BFS).
