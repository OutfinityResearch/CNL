Theory

Controlled language as executable logic.

CNL-PL formalizes natural language into a symbolic representation that can be executed, queried, proven, planned, or optimized without ambiguity.

Core aims

  • Represent constraints and facts in a deterministic AST.
  • Preserve readability while enforcing rigid structure.
  • Enable pragmatic engines to interpret meaning.

Why deterministic language matters

Unambiguous structure

Every valid sentence compiles to a single AST; there is no alternative parse.

Explicit grouping

Boolean logic uses explicit connectors to prevent hidden precedence.

Controlled vocabulary

Determiners and quantifiers separate names from noun phrases.

Pragmatic engines

The surface grammar remains stable while the execution mode changes. Pragmatics define how the AST is interpreted by backends.

Query

Answer structured questions by selecting entities that satisfy conditions.

Proof

Derive whether a proposition is provable, returning supporting steps.

Plan

Generate action sequences using preconditions and effects.

Solve

Resolve unknown variables under explicit constraints.

Design principles

  • Triplet assertions are the atomic unit of truth.
  • Relative clauses replace parentheses for human readability.
  • Explicit connectors remove hidden precedence in logic.
  • Grammar remains stable as pragmatics evolve.
