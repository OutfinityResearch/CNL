# MFOEM (Emotion Ontology)

**Source:** http://purl.obolibrary.org/obo/mfoem.owl
**Format:** OWL
**License:** CC BY 3.0

## Purpose
A structured ontology for emotions, moods, and affective phenomena.

## Key Concepts
- `Emotion` (e.g., Joy, Anger)
- `Mood`
- `AffectiveProcess`

## Usage in CNL
Used in `literature.cnl` for literary criticism and sentiment analysis/tagging of characters and scenes.
