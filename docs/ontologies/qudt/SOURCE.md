# QUDT (Quantities, Units, Dimensions, and Types)

**Source:** http://qudt.org/2.1/schema/qudt
**Format:** Turtle (Mini/Subset)
**License:** CC BY 4.0

## Purpose
Standardized vocabulary for quantities, units of measure, and dimensions.

## Key Concepts
- `Unit` (e.g., Meter, Second)
- `QuantityKind` (e.g., Length, Time)
- `Quantity` (Value + Unit)

## Usage in CNL
Used in `base.formal.cnl` for scientific measurements and physical properties.
